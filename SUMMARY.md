**Endpoint:** `GET /api/lookup?url={domain}`

**Файлы:**
- `server/openrouter-proxy.mjs` (основной endpoint)
- `api/lookup.mjs` (Vercel serverless function)

**Возможности:**
- Мгновенное получение кэшированных результатов
- Скорость: ~50ms vs ~13000ms (в 260 раз быстрее)
- Экономия 100% AI квоты для кэшированных запросов
- Поддержка Redis и локального кэша
- Автоматическая нормализация URL

**Результат:** Пользователи получают результаты мгновенно без повторного AI анализа

## 💡 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Быстрая проверка домена
```bash
curl "https://www.gmotia.tech/api/lookup?url=google.com"
```

### Полный анализ нового домена
```bash
curl -X POST "https://www.gmotia.tech/api/analyze" \
  -H "Content-Type: application/json" \
  -d '{"input":"example.com","telemetryConsent":true}'
```

### Интеграция в JavaScript
```javascript
async function checkUrl(url) {
  const response = await fetch(
    `https://www.gmotia.tech/api/lookup?url=${encodeURIComponent(url)}`
  );
  
  if (response.ok) {
    const data = await response.json();
    return data.result;
  }
  
  // Если не в кэше - делаем полный анализ
  return await analyzeUrl(url);
}
```
