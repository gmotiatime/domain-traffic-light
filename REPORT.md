## 🎯 Примеры использования

### Браузерное расширение
```javascript
async function checkUrl(url) {
  const response = await fetch(
    `https://www.gmotia.tech/api/lookup?url=${encodeURIComponent(url)}`
  );
  
  if (response.ok) {
    const data = await response.json();
    if (data.result.verdict === 'high') {
      alert(`⚠️ Опасная ссылка: ${data.result.summary}`);
    }
  }
}
```

### Batch проверка
```javascript
const domains = ['google.com', 'github.com', 'example.com'];
const results = await Promise.all(
  domains.map(d => 
    fetch(`https://www.gmotia.tech/api/lookup?url=${d}`)
      .then(r => r.json())
  )
);
```
