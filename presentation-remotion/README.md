# Доменный Светофор - Презентация

Видео-презентация проекта для конкурса, созданная с помощью Remotion.

## Запуск

```bash
npm install
npm start
```

## Рендер видео

```bash
npm run build -- DomainLightPresentation out/presentation.mp4
```

## Структура

- 6 слайдов по 5 секунд каждый (30 секунд общая длительность)
- Intro → Problem → Solution → AI Features → Impact → CTA
- Анимации: fade-in, spring scale, плавные переходы
- Разрешение: 1920x1080, 30 fps
