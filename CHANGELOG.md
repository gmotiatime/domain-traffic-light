# Changelog

## 2026-03-30

### Исправления

#### 1. Префилл не работал в расширении
- **Проблема**: Параметр `prefill` передавался в hash-части URL (`#/analyzer?prefill=domain`), но код читал из `window.location.search`
- **Решение**: Обновлена функция `consumeAnalyzerPrefill()` в `src/lib/analyzer-prefill.ts` для корректного парсинга параметров из hash
- **Файл**: `src/lib/analyzer-prefill.ts:18-37`

#### 2. Улучшен дизайн popup расширения
- **Изменения**:
  - Увеличен размер popup: 400px → 420px ширина, 500px → 520px высота
  - Улучшены анимации фоновых orbs с более плавными градиентами
  - Обновлен header: увеличен логотип (32px → 40px), добавлен gradient на заголовок
  - Улучшена glass-card с hover-эффектами и более выраженным backdrop-blur
  - Обновлена кнопка проверки с gradient фоном и shine-эффектом при hover
  - Улучшены карточки результатов с более выраженными тенями и градиентами
  - Обновлен spinner и состояния загрузки
- **Файл**: `extension/popup.html:14-293`

#### 3. Улучшена структура базы данных
- **Проблема**: Возможность дублирования записей для одного хоста
- **Решение**:
  - Добавлено поле `updatedAt` для отслеживания обновлений записей
  - Функция `buildCacheRecord()` теперь сохраняет `createdAt` из существующей записи
  - Функция `setCachedResponse()` проверяет существующую запись перед созданием новой
  - При загрузке БД из файла автоматически удаляются дубликаты (оставляется самая свежая запись)
- **Файлы**: 
  - `server/openrouter-proxy.mjs:247-260` (buildCacheRecord)
  - `server/openrouter-proxy.mjs:291-356` (setCachedResponse)
  - `server/openrouter-proxy.mjs:105-142` (загрузка с дедупликацией)

#### 4. Добавлен скрипт дедупликации БД
- **Новый файл**: `scripts/deduplicate-db.mjs`
- **Функционал**: 
  - Автоматическое удаление дубликатов из `threat-db.json`
  - Создание бэкапа перед изменениями
  - Подробный отчет о процессе дедупликации
- **Запуск**: `node scripts/deduplicate-db.mjs`

### Технические детали

#### Префилл
```typescript
// Было (неправильно):
const urlParams = new URLSearchParams(window.location.search);

// Стало (правильно):
const hash = window.location.hash;
const hashParts = hash.split('?');
if (hashParts.length > 1) {
  const urlParams = new URLSearchParams(hashParts[1]);
  // ...
}
```

#### База данных
```javascript
// Структура записи теперь включает updatedAt:

  key: "...",
  host: "example.com",
  createdAt: 1774887275375,
  updatedAt: 1774887275375,  // ← новое поле
  data: { ... }
}
```
