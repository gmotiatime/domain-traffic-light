## 🗄️ Какая БД используется

Админка использует функции из `server/openrouter-proxy.mjs`:
- `adminCacheGetResponse()` - получение записей
- `adminCacheUpdateResponse()` - обновление
- `adminCacheDeleteResponse()` - удаление

Эти функции уже обновлены для работы с новой БД v2:
- Используют `getRawCacheRecordByHost()` с нормализацией
- Работают с `localDb` (DatabaseManager)
- Поддерживают Redis и memory fallback

## 📝 Изменения в коде

### `src/pages/AdminPage.tsx`

**Добавлено:**
```typescript
// Тип для статистики
type CacheStats = {
  size?: number;
  total?: number;
  active?: number;
  expired?: number;
  verdicts?: Record<string, number>;
  storage?: string;
  dbSize?: number;
  oldestRecord?: string | null;
  newestRecord?: string | null;
};

// State для статистики
const [stats, setStats] = useState<CacheStats | null>(null);

// Функция загрузки статистики
async function loadStats() {
  try {
    const response = await fetch(getApiUrl("/api/cache/stats"));
    const data = await response.json().catch(() => ({}));
    if (response.ok) {
      setStats(data);
    }
  } catch (error) {
    console.error("Failed to load stats:", error);
  }
}
```

**Обновлено:**
- `loadRecent()` - теперь вызывает `loadStats()`
- `handleLogin()` - загружает статистику при входе
- UI - добавлены 4 карточки со статистикой
- Поле поиска - добавлена подсказка о нормализации

## 🚀 Как использовать

### 1. Откройте админку
```
http://localhost:5173/#/admin
```

### 2. Войдите с токеном
Токен из `.env.local`:
```
ADMIN_TOKEN=your_token_here
```

### 3. Смотрите статистику
Сразу после входа увидите:
- Количество записей
- Размер БД
- Распределение по вердиктам
- Информацию о нормализации

### 4. Ищите домены
Можно вводить любой вариант:
- `google.com`
- `www.google.com`
- `https://google.com`
- `https://www.google.com/search?q=test`

Все найдут одну запись!

## 📊 API endpoints

Админка использует:

**GET /api/admin-cache**
- Получение списка записей
- Поиск по host (с нормализацией)

**PATCH /api/admin-cache**
- Обновление записи
- Сохранение правок

**DELETE /api/admin-cache**
- Удаление записи

**GET /api/cache/stats** (новое!)
- Статистика БД
- Размер, количество, вердикты
