# Видео презентация "Доменный светофор.AI"

Видео создано с помощью Remotion для конкурсного проекта по кибербезопасности.

## Описание

30-секундное презентационное видео в стиле сайта с темным дизайном, стеклянными эффектами и плавными анимациями.

### Структура видео (900 кадров, 30 fps = 30 секунд):

1. **Титульная сцена** (0-6 сек) - Главный заголовок с анимированными градиентами
2. **Как это работает** (6-13 сек) - Три этапа: Пауза → Проверка → Действие
3. **Технологии** (13-20 сек) - Четыре уровня защиты
4. **Призыв к действию** (20-30 сек) - Финальный экран

## Установка

Зависимости уже установлены. Если нужно переустановить:

```bash
npm install
```

## Запуск

### Предварительный просмотр в браузере:

```bash
npm start
```

Откроется Remotion Studio на `http://localhost:3000`

### Рендер видео:

```bash
npm run build
```

Видео будет сохранено в `out/video.mp4`

## Настройка

Параметры видео можно изменить в `src/Root.tsx`:

- `durationInFrames` - длительность (900 = 30 сек при 30 fps)
- `fps` - частота кадров
- `width` / `height` - разрешение (по умолчанию 1920x1080)

## Стиль

Видео использует цветовую палитру сайта:
- Фон: черный (#000000)
- Текст: белый с прозрачностью
- Акценты: фиолетовый (#8b5cf6), синий (#3b82f6)
- Статусы: зеленый (безопасно), желтый (внимание), красный (опасно)
