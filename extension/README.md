# Браузерное расширение "Доменный светофор"

Расширение для Chrome и Firefox, которое проверяет безопасность сайтов с помощью AI.

## Возможности

- 🔍 **Быстрая проверка** - проверка текущего сайта в один клик
- 🤖 **AI-анализ** - использует алгоритм с сайта gmotia.tech
- 📊 **Подробный отчёт** - показывает score, вердикт и рекомендации
- 💾 **Улучшение AI** - все проверки сохраняются в БД для обучения алгоритма

## Установка

### Chrome / Edge / Brave

1. Скачайте папку `extension`
2. Откройте `chrome://extensions/`
3. Включите "Режим разработчика" (Developer mode)
4. Нажмите "Загрузить распакованное расширение" (Load unpacked)
5. Выберите папку `extension`

### Firefox

1. Скачайте папку `extension`
2. Откройте `about:debugging#/runtime/this-firefox`
3. Нажмите "Загрузить временное дополнение" (Load Temporary Add-on)
4. Выберите файл `manifest.json` из папки `extension`

## Использование

1. Перейдите на любой сайт
2. Нажмите на иконку расширения 🚦
3. Нажмите "Проверить безопасность"
4. Получите результат анализа

## Конфиденциальность

- Расширение отправляет только доменное имя для проверки
- Данные используются для улучшения AI-алгоритма
- Никакие личные данные не собираются

## Технологии

- Manifest V3
- Vanilla JavaScript
- API: https://www.gmotia.tech/api/analyze
